## Project Overview
This project demonstrates automated testing of a Bakery Management System using Selenium and Python. It focuses on browser automation and functional testing. The system runs on a local environment using XAMPP with a MySQL database.

Tools & Technologies Used

- Python  
- Selenium WebDriver  
- Chrome Browser  
- ChromeDriver  
- XAMPP (Apache & MySQL)  


## Features Tested
### 👤 Customer
- Registration  
- Login  
- Order Placement  
- Payment  
- Invoice Checking  
- Logout  

### 🧑‍💼 Staff
- Login  
- Product Management (Add Product)  
- Order Confirmation  
- Logout  

### 🚚 Delivery
- Login  
- Mark Order as Delivered  
- Logout  


##  Setup Instructions

1. Install Python  
2. Install Selenium  
   ```bash
   pip install selenium


3. Install XAMPP and start Apache & MySQL
4. Place project folder inside htdocs

5. Open browser and run project:
http://localhost/your_project

6. Download ChromeDriver and set it properly

## How to Run Tests

Run any test file using:
python file_name.py

Example:
python test_customer_login.py
